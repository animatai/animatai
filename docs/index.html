AnimatAI¶

AnimatAI is a framework for Articifial General Intelligence developed in Python by Jonas Colmsjö and Claes Strannegård. It is open source and is available at github.com/animatai.

The theory behind the framework is described in:

A General Model for Learning and Decision-Making in Artificial Animals by Claes Strannegård, Nils Svangård, David Lindström, Joscha Bach and Bas Steunebrink

Submitted to IJCAI-17 AGA workshop, Melbourne, Australia

Getting started¶

Install the framework with: pip install animatai

Check out the examples to see how the classes are used.

Demo¶

The examples are hosted at ai.gizur.com. These examples are continuously updated.
